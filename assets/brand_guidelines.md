# Brand Guidelines

## Color Palette
- **Primary**: `#d97706` (Amber/Orange) - Used for branding and borders.
- **Secondary**: `cyan` - Used for micro-animations and status indicators.
- **Background**: `black` - Designed for dark mode terminals.
- **Text**: `white` - Primary content color.
- **Dim**: `grey` - Used for secondary info and reasoning paths.

## Typography
- **Primary**: Monospace fonts (Courier, Menlo, JetBrains Mono).
- **Styles**: Bold headers, italic reasoning blocks.

## Iconography
The CLI uses high-fidelity ASCII characters for the logo:
```text
      *  ▐▛███▜▌ *
     * ▝▜█████▛▘ *
      *  ▘▘ ▝▝  *
```
This represents a stylized diamond/crown structure.
